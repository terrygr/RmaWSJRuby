Create RMA WebService

Here’s how to start:

  1. The url to access it is:

    ~/rma_service/rma/createRma

  2. Action Method:

    POST

  3. The webservice receives the order number and the items to be return in the following format:

    orderNumber = '518402'

    rmaItems = '[ { "sku": "TNF0473-TNFLGIV-ONSI", "price": "14.95", "reason_code": "9", "reason_text": "I didnt like it", "return_quantity": 1 }, { "sku": "TNF0473-TNFLGNV-ONSI", "price": "14.95", "reason_code": "2", "reason_text": "", "return_quantity": 1 } ]'

  4. A successful response would be:

    { "success": 1, "rmaid": 101025816 }

  5. A unsuccesful response would be:

    { "success": 0, "error": "An error ocurred while saving the rma" }

  6. Just a reminder

    This web service will validate that the skus you are returning are available on the order.
